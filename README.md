### How to use my app??

# 1.Download
- Clone with HTTPS, SSH, GitHub CLI
- Download ZIP file
  
# 2. Setup
- Run " npm install " to install all dependence
  
# 3. Start
- Run " npm start " to start my project
- Run " npm run test " to run all my test case
